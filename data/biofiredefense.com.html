BioFire FilmArray BioSurveillance System

Fully Automated BioDetection

The BioFire FilmArray System is able to identify, in a closed system, dozens of the most lethal viruses and bacteria, including emerging infectious diseases. The easy-to-use system represents the next generation in automated detection systems. The BioFire FilmArray System uses a plastic pouch with automated capabilities, including sample preparation, reverse transcription for RNA viruses, and a two-stage nested multiplex PCR process. The results are a revolutionary detection system in a lightweight, small-footprint format.

BioFire FilmArray Test Kits. 1 Streamlined System. Many BioThreat Targets.

The BioFire FilmArray panels contain all the required reagents for sample preparation, PCR, and detection in a simple freeze-dried, room temperature stable format making it the ideal system for any mobile lab. Prior to a run, the operator injects the unknown sample and hydration solution into the pouch and the FilmArray instrument does the rest.

RAZOR Mk II BioDetection System

The RAZOR Mk II System detects and identifies biological agents using cutting edge PCR technology. Made for field use, RAZOR Mk II is compact, lightweight, and easy to use. It is a field PCR unit that is fast, with ultra-reliable DNA based results. Created for first responders and front line military troops, it is easily operated while working in protective equipment under extreme conditions. The new battery powered unit includes barcode reader, and a bright, easy to read color screen.
